eBUS SDK .NET Code Samples

The following sample programs are provided to help illustrate various features of the .NET API for the eBUS SDK. All samples are written in C# unless otherwise noted. Before changing the code for any of the samples, Pleora recommends that you copy the sample from the installation directory to your personal Visual Studio work space.

Sample Description
PvDualSourceSample Shows how to control and stream from a dual source GigE Vision device.
PvGenBrowserWndSample Shows how to use a PvGenBrowserWnd GenICam UI to control parameters.
PvGenParameterArraySample Shows how to use the PvGenParameter of PvGenParameterArray.
PvMulticastMasterSample Shows how to configure a GigE Vision device to stream to a multicast group.
PvMulticastSlaveSample Shows how to join a multicast group and receive a stream.
PvPipelineSample Shows how to use the PvPipeline class for image acquisition.
PvPipelineSampleVB Shows how to use the PvPipeline class for image acquisition (written in Visual Basic).
PvSimpleUISample Shows a typical simple eBUS SDK application.
PvSimpleUISampleVB Shows a typical simple eBUS SDK application (written in Visual Basic).
PvTransmitTestPatternSample Shows how to transmit a test pattern stream to a multicast group.
PvTransmitTestPatternSampleVB Shows how to transmit a test pattern stream to a multicast group (written in Visual Basic).
PvTransmitTiledImageSample Shows how to take four incoming receive streams and combine them into a single, tiled transmit stream.
