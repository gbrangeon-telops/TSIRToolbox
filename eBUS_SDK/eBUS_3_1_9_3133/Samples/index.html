eBUS SDK C++ Code Samples

The following sample programs are provided to help illustrate various features of the C++ API for the eBUS SDK. Before changing the code for any of the samples, Pleora recommends that you copy the sample from the installation directory to your personal Visual Studio work space.

Sample Description
eBUS Driver Installation API Shows how to display information about network configuration and apply eBUS drivers to network adapters on a given PC.
GEVPlayerSample Contains the source code for the GEVPlayer application which is used to detect, connect and configure a GigE Vision devices.
NetCommandSample Contains the source code for the NetCommand application which is used to detect, connect and configure multiple GigE Vision devices simultaneously.
PvBufferWriterSample Illustrates how to use the PvBufferWriter and PvBuffer classes.
PvCamHeadSerialComLogSample Illustrates how to create a simple UI application providing basic device control and streaming capabilities, and how to monitor serial communications.
PvConfigurationReaderSample Illustrates how to use the PvConfigurationReader and PvConfigurationWriter classes.
PvDeviceFindingSample Illustrates how to use the PvSystem, PvInterface, PvDeviceInfo and PvDeviceFinderWnd classes to detect and enumerate GigE Vision devices.
PvGenParameterArraySample Shows how to use the PvGenParameter of PvGenParameterArray.
PvMulticastMasterSample Shows how to configure a GigE Vision device to stream to a multicast group.
PvMulticastSlaveSample Shows how to join a multicast group and receive a stream.
PvMultiSourceSample Shows how to use the PvPipeline class for image acquisition from a multi source device.
PvPipelineSample Shows how to use the PvPipeline class for image acquisition.
PvPlcAndGevEvents Shows how to control and handle PLC states and handle GigE Vision events.
PvReceiveRawSample Illustrates how to use the PvStream and PvPipeline classes to receive raw data using the eBUS SDK.
PvRecoverySample Illustrates how to create a simple command line application performing typical image acquisition that supports automatic recovery.
PvRGBFilterSample Shows how to select a device, grab an image, convert to RGB32, apply various filters and save to disk.
PvSerialBridgeSample Shows how to use the PvSerialBridge class control a serial device connected to a GigE Vision IP Engine.
PvSerialPortIPEngineSample Shows how to use the PvSerialPortIPEngine class for serial communication through a GigE Vision IP Engine.
PvSimpleUISample Shows a typical simple eBUS SDK application.
PvStreamSample Shows how to use the PvStream class for image acquisition.
PvTransformAndTransmitSample Illustrates how to receive video from a GigE Vision device, resample it, apply a text overlay, and retransmit.
PvTransmitRawSample Illustrates how to use the PvVirtualDevice and PvTransmitterRaw classes to transmit raw data (not images).
PvTransmitScreenSample Illustrates how to use the PvVirtualDevice and PvTransmitterRaw classes to transmit the contents of the screen.
PvTransmitTestPatternSample Illustrates how to use the PvVirtualDevice and PvTransmitterRaw classes to transmit a test pattern.
PvTransmitTiledImageSample Shows how to take four incoming receive streams and combine them into a single, tiled transmit stream.
